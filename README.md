# Distros Linux

Site informativo desenvolvido em HTML e CSS com o objetivo de apresentar diversas distribuições Linux, seus requisitos, características, ambientes de desktop e informações sobre jogos no Linux utilizando Proton.

Acesse o site: https://nicolasfmp.github.io/Distros-Linux/

## Sobre o Projeto

O Distros Linux é um portal estático criado para auxiliar usuários interessados em conhecer o ecossistema Linux. O site reúne informações sobre distribuições populares organizadas por categorias, ambientes de desktop e ferramentas utilizadas para jogos.

**Categorias de distribuições incluídas**


- Iniciantes / Estáveis

  - Ubuntu
  - Debian
  - Mint
  - Zorin OS
  
- Gaming / Performance

  - CachyOS
  - Nobara
  - Bazzite
  - SteamOS

- Produtividade / Trabalho

  - Fedora
  - Pop!_OS

- Segurança / Privacidade
    
  - Qubes OS
  - secureblue
  - Tails
  - Kali
 
- Leve / Computadores fracos

  - antiX
  - Q4OS
 
- Avançadas

  - Arch
  - NixOS
    
## Funcionalidades

- Apresentação de distribuições Linux organizadas por categorias.
- Exibição de requisitos de hardware.
- Capturas de tela das distribuições e ambientes de desktop.
- Explicação sobre o kernel Linux e o projeto GNU.
- Guia sobre Proton para execução de jogos no Linux.
- Links oficiais para download das distribuições.
- Navegação rápida por meio de menus e âncoras internas.

## Instalação

**Executar localmente**

1. Clone o repositório
```text
git clone https://github.com/NicolasFMP/Distros-Linux.git
```
2. Entre na pasta
```text
cd Distros-Linux
```
3. Abra o arquivo `index.html`

**Windows**
```text
start index.html
```
**Linux**
```text
xdg-open index.html
```
**macOS**
```text
open index.html
```

**Utilizando o VS Code**

1. Abra a pasta do projeto no VSCode
2. Instale a extensão: `Live Preview` pela Microsoft
3. Clique com o botão direito no arquivo `index.html` e pressione: `Show Preview`
4. Clique no menu hamburguer na tela de preview e selecione: `Open in Browser`
