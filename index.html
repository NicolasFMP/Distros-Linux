O que é Linux?

O Linux, de forma mais avançada, não é exatamente um sistema operacional completo sozinho. Tecnicamente, o Linux é apenas o kernel, ou seja, o núcleo do sistema. Esse kernel foi criado por Linus Torvalds em 1991 e é responsável por fazer toda a comunicação entre o hardware do computador e os programas que você usa. Ele gerencia processos, memória, dispositivos e garante que tudo funcione ao mesmo tempo sem conflitos.

Antes mesmo do Linux existir, já havia um grande projeto chamado GNU Project, iniciado por Richard Stallman em 1983. O objetivo do GNU era criar um sistema operacional totalmente livre, inspirado no Unix. Esse projeto conseguiu desenvolver quase tudo: compiladores, bibliotecas, shell e vários comandos essenciais que usamos até hoje, como ls, cp, rm e muitos outros. O que faltava era justamente um kernel funcional e estável para completar o sistema.

Quando o kernel Linux ficou pronto, ele foi combinado com as ferramentas do GNU. Essa junção criou um sistema operacional utilizável na prática. Por isso, tecnicamente, muitos defendem o nome “GNU/Linux”, já que o sistema não é apenas o kernel, mas também todo o conjunto de ferramentas que fazem o ambiente funcionar.

As distribuições Linux, como Ubuntu, funcionam como um “pacote completo” que junta o kernel Linux, as ferramentas do GNU, bibliotecas essenciais, gerenciador de pacotes, interface gráfica e aplicativos. Outras distribuições, como Arch Linux, seguem uma abordagem mais minimalista, deixando que o usuário construa o sistema do zero conforme suas necessidades.

No fundo, quando alguém diz que usa “Linux”, geralmente está usando um conjunto de tecnologias: o kernel Linux, as ferramentas do GNU, o shell como o Bash, o sistema de inicialização como systemd e diversos outros componentes que juntos formam um sistema operacional completo e funcional.

Distribuições Linux por categorias

Iniciantes / Estáveis

Ubuntu

Download

Ubuntu é um sistema operacional composto de software livre e de código aberto e de software proprietário na forma de blobs, construído a partir do núcleo Linux, baseado no Debian e utiliza GNOME como ambiente de desktop de sua mais recente versão com suporte de longo prazo (LTS). Esta distribuição Linux é desenvolvida pela Canonical Ltd. Geralmente é executado em computadores pessoais e também é popular em servidores de rede, geralmente executando a versão Ubuntu Server, com recursos de classe empresarial. Até 2017, o Ubuntu também estava disponível para tablets e smartphones, com a edição Ubuntu Touch.

CachyOS

Download

CachyOS é uma distribuição Linux baseada no Arch Linux. Lançado em 2022, tem o seu foco na otimização. segurança do sistema e um suporte melhor para máquinas mais modernas. Também é altamente personalizável com vários ambientes de Desktop como GNOME, KDE Plasma 6, Xfce, i3, LXQt, Cinnamon, LXDE, MATE, Hyprland e entre outros mais.

Nobara Linux é uma distribuição Linux lançada em 10 de julho de 2022 baseada no Fedora Linux . É desenvolvido por Thomas Crider, também conhecido coloquialmente como "Glorious Eggroll", que também é conhecido por desenvolver o Proton-GE, uma modificação da camada de compatibilidade da Valve Corporation , amplamente utilizada para executar jogos do Windows no Linux.
